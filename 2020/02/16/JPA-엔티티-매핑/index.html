데이터베이스 스키마 자동 생성

DDL 을 애플리케이션 실행 시점에 자동생성
테이블 중심 -> 객체 중심
데이터베이스 방언을 활요해서 적절한 DDL 생성
생성된 DDL 은 개발 장비에서만 사용.

아래의 JPA 옵션을 적용하여 실행할 경우.

옵션 기능
